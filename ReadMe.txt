Polyfacetool example demonstrates creation of and conversion to mesh element using Polyface API

This example provides two tools

POLYFACE CREATE			: Shows how to create a mesh with only one triangle facet. 
POLYFACE CONVERT		: Shows how to convert an existing element to a mesh. This tool also shows how to use IElementGraphicsProcessor.
